Ankohl Ruins (Part I)


 Ankohl Ruins Outskirts



If you wish to enter this forgotten tower northeast of Champa you should have beaten the Gaia Rock Dragon Oorochi.

Only then Felix will have the ability to use SAND psynergy. This is neccessary to get past many sandy obstacles which are awaiting you inside the tower.

So, if Felix does not know SAND, sail to Izumo in the far east, enter Gaia Rock (south of Izumo), fight Oorochi and finally return here.

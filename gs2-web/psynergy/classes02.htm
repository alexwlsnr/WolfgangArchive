Character Classes


One of the nicest features within Golden Sun is the character class management. While in the first adventure of Isaac the classes were strictly bound to the different djinn ally combinations, in the sequel three new classes are to be found, which depent on a special item. For details refer to the following tables.

The lists are far from compeltion, I simply have no time to test all the different possible combinations and the translations of the class names are very buggy, too. Maybe you will complete the classes tables when the english version hits the market.

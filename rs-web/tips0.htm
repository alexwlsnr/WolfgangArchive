Tips for Playing The Royal Stone (Start)
Press the key defined as the START BUTTON (RETURN or SPACE, depends on the Emulator you are using).

A new menu will appear. Here you can choose out of four different actions. Those of you who like to play the game and have no experience with the Japanese language should print the frame on a printer and refer to the sheet when playing. On the left you always see the original Japanese menu, on the right you find the English translation.

Use the A button to confirm, B is used to cancel the command (the key you have to press depends on the emulator you use. Normally they are RIGHT CRTL for A and ALT for B).

Start Menu
 Start Menu (Translation)
Above you see the menu displayed when playing ´The Royal Stone´ for the first time.

If you choose Start the intro of the game shows what happens to Eva at the beginning. Skip the scene by pressing START again. The first battle will start. The menus below are available only if you have saved a game at an inn or within the Command Menu.

Continue Option
 Translation
Use Continue to start from the last saved position. You can choose out of three different games (if saved before).
Copy Option
 Translation
You can copy a saved game to a different save slot. If the slot is not free, the old saved game will be replaces by the copy.
Delete Option
 Translation
The last option is Delete a saved game. You have to confirm your selection. Be careful not to delete a game you want to continue (I know that you know that, too. It´s only that I like writing so much).

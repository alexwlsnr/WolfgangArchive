Tips how to play


Before we start with the game, look below to see how the menues could have looked like, if the game had been published in English. If you want to play this game, refer to them for an easier gameplay.

Game Start

Start a new game of load a saved one. You can choose from three previously saved files.


Battle Menu

There are more commands available than you can see on the screen.

You have to scroll through the menu list to display the rest!

The chosen command is marked by a red underline.

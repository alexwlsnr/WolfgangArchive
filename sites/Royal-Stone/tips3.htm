Tips for Playing The Royal Stone (3)
If you finally want to leave the town or village answer Yes to Eva´s question and a new Command screen will be shown. The first option within this menu is Status.
Command Status
 Command Status (Translation)
After you selected the Status option, a list of all your heros (and tamed monsters) will appear.
Choose an ally and his/her status will be displayed. By press Button B you will get information about the magic abilities, if the chosen character can use magic.

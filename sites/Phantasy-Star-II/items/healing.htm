Items


No Role Playing Game would be that fun if you can't buy new (and better) weapons, find special items, trade rare things, etc.

In this section you will not find detailed statistics about all items which can be found during the adventure but only a rough overview what is available. Most pictures are original scans of the manual which can be found in the first Phantasy Star Collection, published for the Sega Saturn.
